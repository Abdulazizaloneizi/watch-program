 Smart Watch Pickup System (Java Prototype)

## Features
- Parent watch simulation
- Child watch simulation
- School relay server
- QR-based pairing
- Wi-Fi socket communication
- Timer + conflict alarm system
- Watch-style UI (JavaFX)

## Run order
1. Run SchoolServer
2. Run ParentWatch
3. Run ChildWatch

## Libraries required
- JavaFX SDK
- ZXing Core
- ZXing JavaSE
- ZXing cam

## Notes
- GPS is simulated
- Wi-Fi uses a localhost socket
- Designed as a hardware-ready prototype
